1. Resolved issues

- Differential expression searches gave an unhelpful error message if a non-gene string was entered.

- Several classes were removed from the source distribution, which were inadvertantly 
included in the individual release.

2. Known issues

- No new

3. Installing and running Gemma at your location

Gemma is designed to be portable. However, we have not yet completed
documenting the build and installation procedure. If you are
interested in installing Gemma, please contact gemma@bioinformatics.ubc.ca.

4. Downloading data

Many of the pages in Gemma have links for downloading data of various
types. Downloads of more complete data sets representing the Gemma
database will be made available soon.

5. Contacting us

Please email gemma@bioinformatics.ubc.ca with bug reports, feature
requests, or other feedback.
