1. Resolved issues and improvements

- Revised gene chooser interface allows multiple genes to be pased from clipboard.

- Coexpression search results shows the data sets providing support for each gene.

- Coexpression search form allows viewing all the data sets used in the analysis.

- Gene chooser and multiple genes supported for differential expression searches.

- Performance improvements for many operations.

- Download links for data matrices and experimental designs.

- Many other bug fixes and behind-the-scenes changes and improvements.

2. Known issues

- The coexpression search page does not display correctly in Internet Explorer for Windows, version 6 or earlier 
(though it is functional for the most part). We recommend IE users upgrade to version 7.

3. Installing and running Gemma at your location

Gemma is designed to be portable. However, we have not yet completed
documenting the build and installation procedure. If you are
interested in installing Gemma, please contact gemma@bioinformatics.ubc.ca.

4. Downloading data

Many of the pages in Gemma have links for downloading data of various
types. Downloads of more complete data sets representing the Gemma
database will be made available soon.

5. Contacting us

Please email gemma@bioinformatics.ubc.ca with bug reports, feature
requests, or other feedback.
