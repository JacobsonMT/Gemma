Improvements and new features

Gemma 1.7 is a major new release with numerous new features and improvements, too many to list here. Highlights include

- Revamped web interface
- New differential expression and coexpression interfaces, including new visualizations

Bug fixes

Much of the focus for the release was stability and performance. Hundreds of issues have been closed.
