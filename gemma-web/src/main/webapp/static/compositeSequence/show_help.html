Composite Sequence (Probe) Details Help

Displays details for a given composite sequence (probe) found on an platform


  • Name: Identifier for the probe, provided by the manufacturer or the data submitter
  • Description for the probe, usually provided by the manufacturer. It might not match the sequence annotation!
  • Sequence Type: The type of this sequence as recorded in our system
  • Sequence Name: Name of the sequence in Gemma
